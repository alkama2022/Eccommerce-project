# 🛒 E-commerce Project (Django)

A full-featured eCommerce web application built with Django. This project provides core online shopping functionality including product browsing, cart management, and order processing.

---

## 🚀 Features

- 🧑‍💻 User Authentication (Register, Login, Logout)
- 🛍️ Product Listing & Detail Pages
- 🛒 Shopping Cart System
- 💳 Order Management
- 🛠️ Admin Dashboard for managing products and users
- 🐞 Debug Toolbar for development insights

---

## 🧰 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap (optional)
- **Database:** SQLite (default)
- **Tools:** Django Debug Toolbar
